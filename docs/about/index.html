About

鲁迅说过：写作总是蠢蠢欲动 麦克阿瑟表示：不是独立博客不好，而是 GitHub 更具性价比 懒得回家在树下睡觉的牛顿总结出：懒，才是第一生产力

所以，这是一个自动化博客，无需任何依赖安装，只需要专注写博客内容

  • 博客内容存放在 GitHub issues 上，不担心数据丢失
  • 可自定义博客模板，满足个性化需求
  • 自动化，更新 issue 则自动更新博客内容，任何设备都可以写博客
  • 可多人协作，指定作者，非作者写的 issues 会自动排除

总之，可专心写博客，不需要关注技术细节，所有都自动完成
