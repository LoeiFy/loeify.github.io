谈谈我的妻子

偶然刷 v2 帖子看到有人发帖描述自己老婆属于贤妻良母类型，并列举了老婆的性格特点。原帖：https://v2ex.com/t/1167927

看到此，略有所思，感觉我也可以照着列一下自己老婆的情况，参照帖子的格式

优点

  1. 皮肤白皙，长相清秀，身材苗条，身高 162 ，两小孩还小，别人看到各种夸长得真好，白白胖胖
  2. 学历本科，广州 211，毕业就在国企事业单位
  3. 情绪稳定，会发脾气但好谈话，能讲道理，有问题直接说
  4. 跟我父母同住偶尔吐槽，但小情况基本都能让步，但某些例如不让我爸在屋里吸烟就坚决不让
  5. 母性一般强？爱小孩，但并不围着小孩转。如果让她可以去玩不带小孩，她会很开心
  6. 据我观察，她工作认真负责，现在中层负责人，不过他也经常吐槽上班不易
  7. 经济独立，我两各自管钱，我负责大的支出，她负责生活的，小孩的，外面吃也基本她付钱。偶尔自己买个包也不需要
  8. 大事都要征求我的意见，共同决策事情
  9. 为家庭着想，结婚彩礼事情跟她妈吵起来说要太多了，叫我不要给太多

缺点

  1. 不爱整理，各种东西乱放，基本不收拾，都是我整理
  2. 做事不够坚持，例如突发奇想说学法律相关，买了一堆资料，但后面没有坚持忘记了
  3. 做饭家务活一般，不过这个不算什么缺点，我也没什么要求
  4. 喜欢刷剧，有时候看着控制不了时间，但也还好，不会熬夜刷，该睡觉就睡觉

总结：我觉得已经很好了，不奢求其他
